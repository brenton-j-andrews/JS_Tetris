Tetris

Desktop Controls:

Left and Right Arrows:

Navigate level selection page. Shift pieces left and right in game.

Down Arrow:

Navigate level selection page. "Soft drop" a piece in game. Hard drop not available.

Up Arrow and Z:

Rotate pieces clockwise and counterclockwise respectively.

Enter:

Start game from both selection and game over pages.

Backspace:

Return to selection page from game and game over pages.

Mobile Controls:

Note: Due to lack of tactile buttons, the on screen buttons are hard to use.

Left and Right Arrows:

Same as desktop

Down Arrow:

Same as desktop

B and A buttons

Rotate pieces clockwise and counterclockwise respectively.

Start:

Start game from both selection and game over pages.

Select:

Return to selection page from game and game over pages.
